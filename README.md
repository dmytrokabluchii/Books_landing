Single page landing, responsive design.

Technologies used: HTML, SCSS, Pure Java Script, AJAX, Axios, Telegram bot, SmtpJS, Swiper.js, Gulp.

To start gulp project:
<br>
npm install --global gulp-cli
<br>
npm init
<br>
npm install --save-dev gulp
<br>
gulp build (if appear error, to start "gulp build" second time!)
<br>
gulp serve
